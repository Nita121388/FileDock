# 2026-02-01

## Phase 7 (Browse/Restore MVP)

- Added snapshot browse endpoints:
  - `GET /v1/snapshots/:snapshot_id/tree?path=`
  - `GET /v1/snapshots/:snapshot_id/file?path=...`
- Added CLI commands:
  - `filedock tree` (list directory)
  - `filedock pull-file` (download one file)
- Updated API/build docs.

## Phase 8 (Multi-Chunk + Pull-Folder)

- Manifest supports multi-chunk files (backward compatible with legacy `chunk_hash`).
- Client uploads files in fixed-size chunks (MVP: 4MiB chunks).
- Server reconstructs files from multiple chunks for `GET /file`.
- Added CLI command: `filedock pull-folder` (restore entire snapshot).
