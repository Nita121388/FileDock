# GitHub Access (Notes)

This repo may be private. To clone/push from a machine without interactive GitHub login:

## Option 1: Repo Deploy Key

- Generate a dedicated SSH key per repo.
- Add the public key to the repo: Settings -> Deploy keys.
- Recommended when you want least privilege.

## Option 2: Account SSH Key

- Generate one SSH key for the GitHub account.
- Add the public key to the account: Settings -> SSH and GPG keys.
- Recommended when you work across multiple repos.
